# Product Context

## Purpose
To provide advanced graph analysis capabilities for Obsidian notes, helping users understand symmetric relationships and importance of different notes in their knowledge base.

## Problem Solution
- Calculates various centrality metrics using undirected graph analysis
- Provides insights into symmetric note relationships
- Helps users identify key documents and bidirectional connections
- Ensures thread-safe operations for reliable analysis

## Core Functionality
1. Graph Analysis
   - Calculate symmetric degree centrality for connection analysis
   - Calculate symmetric eigenvector centrality for influence analysis
   - Calculate symmetric betweenness centrality for bridge document identification
   - Calculate symmetric closeness centrality for accessibility analysis

2. Integration
   - Seamless integration with Obsidian's existing graph visualization
   - Efficient processing of symmetric note relationships
   - Thread-safe operations for reliable results
   - Clear presentation of metrics

## User Experience Goals
1. Fast and efficient calculation of symmetric metrics
2. Reliable and thread-safe operations
3. Minimal resource usage
4. Seamless integration with existing Obsidian workflow

## Current Focus
Optimizing symmetric relationship analysis while ensuring thread safety and maintaining all core functionality
